# 📌 JSP 게시판
    ◼ 개발 기간 : 2023.08.01 ~ 2023.08.11 (11일)
    ◼ 기능      : 이용약관/회원가입/로그인/게시판&댓글CRUD)
                + 회원가입 시 이메일 인증 기능
                + 아이디/비밀번호 찾기 기능
                + 마이페이지에서 회원정보 수정 기능
    ◼ 개발환경  : Eclipse | Java 17 | JSP 2.3 | Tomcat 9 | MySql 8
